<div align="center" >
</br>
LoadStar는 개발자가 꿈인 사람들이 자신의 진로 방향을 찾기위해 여러 사람들의 공부 그래프를 보며, 소통하고 참고하며, 어떤 이에게는 자신의 길을 보여주며 같은 방향을 걸어가고, 또 다른 이에게는 그 길을 바탕으로 새로운 지도를 만들어가는 웹 사이트 입니다.

</br>
</br>
